linux 虚拟机 ip配置

2021-01-18 linux linux, 虚拟机 评论

VMware创建linux虚拟机后，未设置网络不能上网

  • 选用NAT模式

  • 主界面中点击 编辑>虚拟网路编辑器

    • 设置网段：192.168.178.0
    • 子网掩码：255.255.255.0
    • 设置子网IP： 192.168.178.0
    • 关闭DHCP

  • 配置 “NAT设置”

    • 设置网关：192.168.178.254

  • 设置完，即可重启虚拟机进行安装

  • 在Centos7 设置IP

    • 方法一： 自动分配IP

      • 设置界面：Network&Hostname点击 “ON”（开启DCHP才能自动分配ip）

    • 方法二：手动分配IP

      • ip addr show

      • vi /etc/sysconfig/network-scripts/ifcfg-ens33

        • 重启网络服务
